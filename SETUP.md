# Setup Instructions for Times of NYTVNT

## Step 1: Install Dependencies
Open terminal in the `times-of-nytvnt` folder and run:
```bash
npm install
```

## Step 2: Test the Application
After installation completes, run:
```bash
npm run dev
```

Then open your browser to `http://localhost:5173` to see the news website in action!

## Step 3: Push to GitHub (After Testing)
Once you've verified everything works, run these commands:

```bash
git init
git add .
git commit -m "Initial commit - Times of NYTVNT news website"
git branch -M main
git remote add origin https://github.com/pred07/Times-OF-NYTVNT.git
git push -u origin main
```
